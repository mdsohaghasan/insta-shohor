Question :
how does work javascript

Answer:
Javascript একটি সিঙ্গেল থ্রেড প্রোগ্রামিং ল্যাঙ্গুয়েজ । এটি ইভেন্ট লুপ করে সিনক্রোনাস ও অ্যাসিনক্রোনাস দুই ভাবে কাজ করে । Javascript এক্সিকিউট করে v8 ইঞ্জিন । এই V8 ইঞ্জিন Javascript এর কোডগুলোকে ইন্টারপ্রেট করে JIT (Just-in-Time) কম্পাইলার ইউজ করে একই সাথে ফাস্ট লি ও DEBUG ফ্রেন্ডলি করে এক্সিকিউট করে । ব্রাউজার এর মধ্যে Javascript গুলোকে ব্রাউজার parser RENDER TREE বা সিনট্যাক্স কোড বানায় তারপর V8 ইন্টারপ্রেটার ইঞ্জিন সেটাকে সিনট্যাক্স কোড থেকে বাইক কোড বানায় তারপর v8 turbofan compiler সেটাকে মেশিন কোড বানায় । এভাবেই জাভাস্ক্রিপ্ট কাজ করে

Question :
what is event loop in javascript

Answer :
Javascript সাধারণত একটি কোড ব্লক এর মধ্যকার কোডগুলো তে সিক্যুয়েন্সলি বা সিনক্রোনাস পদ্ধতিতে কাজ করে এবং কিন্তু settimeout fetch এ জাতীয় এসিনক্রোনাস টাস্ক গুলোকে হ্যান্ডেল করার জন্য তারা একটি সহকারী ব্যবহার করে । সেটি হচ্ছে web api বা c++ api । তারপর CALL STACK যখন তার সবগুলো টাস্ক কমপ্লিট করে ফ্রি হয়ে যায় তখন web api বা c++ api এ থাকা টাস্কগুলো টাইমার শেষ হওয়া অনুযায়ী QUEUE এর মধ্যে দিয়ে দেয় । সেখানে CALL STACK ও QUEUE মধ্যে যোগাযোগ রক্ষা করার জন্য একটি ইভেন্ট লুপ কাজ করে । ইভেন্ট লুপ তখন QUEUE থেকে ফাস্ট ইন ফাস্ট আউট এই হিসেবে QUEUE এর কাছে থাকা প্রত্যেকটি টাস্ক দিয়ে দেয় CALL STACK এর কাছে । CALL STACK তখন সে গুলোকে সিঙ্গেল থ্রেড অনুযায়ী এক্সিকিউট করে ।
